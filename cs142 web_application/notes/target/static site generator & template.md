## 静态网站生成器SSG

### 概述

**定义**

静态网站生成器是一种基于原始数据和一组模板生成完整静态 HTML 网站的工具。

**作用**

从本质上讲，静态站点生成器自动完成对单个 HTML 页面进行编码的任务，并让这些页面提前准备好为用户提供服务。

因为这些 HTML 页面是预先构建的，所以它们可以在用户的浏览器中非常快速地加载。
